# O Que tens de fazer

  - abrir o drop down RUN
    - Dentro do Run ir a Edit Configurations
        - Nas configurations Procurar os Program Arguments
        ```sh
            - [0] = URL que faz o download
            - [1] = O numero iteracoes que queres no loop
        ```
